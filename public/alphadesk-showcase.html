The Problem

Every morning, PMs waste 2.5 hours on manual prep.

Discretionary portfolio managers at hedge funds and family offices spend the first half of every trading day doing the same low-value work before any actual analysis can begin. AlphaDesk eliminates this entirely.

Step 01
Price & Performance Pull
Manually export EOD prices, compute returns, calculate Sharpe, Sortino, max drawdown across every position.
~30 min daily
Step 02
News & Filing Scan
Scan Bloomberg, Reuters, SEC EDGAR for overnight 8-Ks, earnings releases, and analyst upgrades per holding.
~45 min daily
Step 03
Risk Limit Review
Cross-reference portfolio metrics against VaR, volatility, and drawdown limits. Identify anything breached overnight.
~20 min daily
Step 04
Correlation Check
Recalculate pair-wise correlations and beta exposures, flag concentration risk and factor cluster drift.
~25 min daily
Step 05
Research Synthesis
Read analyst notes, pull fundamental metrics via Bloomberg or FactSet, write a brief for the morning meeting.
~40 min daily

Business Objective

  • Reduce pre-market prep from 150 minutes to under 15 minutes
  • Surface only decision-relevant signals — no noise, no manual triage
  • Enforce risk limits automatically with human approval for breaches
  • Deliver a PM-ready tearsheet and client letter before market open
  • Provide full source attribution — every number is auditable
  • Run end-to-end for $0.23 per session (vs. $40k/year Bloomberg terminal)

End-to-End Workflow

5:00 AM to 8:50 AM. Fully autonomous.

AlphaDesk runs the full pre-market sequence while the PM sleeps. By the time they arrive, the brief is waiting. Human input is required only at the risk gate — every other step is agent-driven.

  • 5:00 AM
    Agent 1 — Data Ingestion
    Market Data Pull
    yfinance pulls EOD prices for all portfolio tickers and SPY benchmark. FRED API fetches the current DTB3 risk-free rate. Data is validated, cleaned, and cached in the session DATAFRAME_CACHE.
  • 5:08 AM
    Agent 2 — Metrics
    Performance Analytics
    Computes 14 metrics: annualised return, volatility, Sharpe, Sortino, max drawdown, Calmar, VaR 95%, CVaR, Omega ratio, beta, information ratio, Treynor, attribution, and drawdown series.
  • 5:18 AM
    Agent 3 — Fundamental Research
    ReAct Research Loop
    LangChain ReAct agent calls FMP API tools iteratively — pulling P/E, EV/EBITDA, revenue growth, debt ratios, and analyst estimates for each holding. Synthesises findings with GPT-4o.
  • 5:35 AM
    Agent 4 — Signal Monitor
    Multi-Source Signal Aggregation
    Pulls signals from 6 sources: SEC EDGAR (8-K/10-Q), Finnhub news, Reuters RSS, Reddit PRAW, Stocktwits, Alpha Vantage. Applies 3-tier credibility scoring and deduplication before ranking.
  • 6:10 AM
    Agent 5 — Risk Monitor · HITL Gate
    Risk Limit Enforcement
    Evaluates portfolio against drawdown (15%), volatility (25%), and VaR (2.5%) limits. If any limit is breached, LangGraph interrupts execution and sends an alert. PM must approve before the graph resumes.
  • 7:00 AM
    Agent 6 — Backtesting
    Strategy Validation
    vectorbt runs the configured strategy backtest over 2-year history. GPT-4o narrates results — explaining what worked, what failed, and whether current conditions resemble past drawdown regimes.
  • 7:30 AM
    Agent 7 — Correlation
    Concentration & Factor Risk
    Scipy computes the full pairwise correlation matrix and individual beta to SPY. Alerts fire at 0.75 threshold. Identifies factor clusters and concentration risk across the portfolio.
  • 8:50 AM
    Agent 8 — Report Generator
    PM Tearsheet + Client Letter
    GPT-4o synthesises all upstream outputs into a full PM tearsheet and a client-ready letter. WeasyPrint renders the tearsheet as a PDF. All sources cited. Delivered before market open.

Signal Intelligence

3-tier credibility. No noise.

Signal sources are weighted by institutional credibility — not by volume. SEC filings and verified news anchor every analysis. Retail social sentiment provides context, never conclusions.

Tier Sources Credibility Weight Role Access
Tier 1 SEC EDGAR (8-K, 10-K, 10-Q) · Finnhub Market News 1.0× Primary signals — always included in analysis Free · No rate limit (SEC) · 60 req/min (Finnhub)
Tier 2 Reuters RSS Feed · Alpha Vantage News 0.9× Professional wire — geopolitical and macro context Free · Public RSS (Reuters) · 25 req/day (AV)
Tier 3 Reddit PRAW (r/WSB, r/investing, r/stocks) · Stocktwits 0.5× Retail sentiment context only — labelled, never primary Free · Reddit OAuth (100 QPM) · Stocktwits public API

Note: X/Twitter API Basic = $100/mo with severe rate limits. X API Standard commercial = $12,000/year. Not viable for a prototype. In production, X API monitors verified policy accounts. For this prototype, the above stack provides equivalent coverage at $0/month.

Human-in-the-Loop

Agents execute. Humans decide.

AlphaDesk enforces a mandatory human approval gate for risk-sensitive decisions. This is not a UI pattern — it is a LangGraph interrupt_before node that literally pauses graph execution until a PM approves or overrides.

⚠️
LangGraph Human-in-the-Loop Gate

When the Risk Monitor agent detects a limit breach, interrupt_before=["risk_monitor"] halts the LangGraph state machine. The graph cannot proceed — to Report Generator, to Backtest, or to any downstream node — until the PM explicitly resumes with human_approved=True.

Step 01
Limit Detected
Risk Monitor computes metrics and finds a breach. Sets human_approval_needed=True in state.
Step 02
Graph Interrupted
LangGraph interrupt_before fires. Execution pauses. MemorySaver checkpoints current state.
Step 03
PM Notified
Alert is surfaced in the Risk Monitor UI with full context — which limit, by how much, and the primary position driver.
Step 04
PM Decision
PM approves, overrides, or halts. Decision is injected into graph state. LangGraph resumes from checkpoint.
Security & Guardrails
🔒
3-Stage Content Filter
Every LLM output passes through run_content_filter() before reaching any user or downstream node. Financial content safety, hallucination risk flagging, and investment advice guardrails.
📋
Source Attribution
Every number is tagged with its source. AuditTooltip components surface data provenance (yfinance, FMP, FRED) directly in the UI — PM knows what to trust and why.
🔍
LangSmith Tracing
All agent runs are fully traced in LangSmith — every node input/output, token count, latency, and tool call. Debuggable, auditable, and demoable to judges.
⚡
Redis Rate Limiting
API calls are rate-limited via Redis to respect source limits (60 req/min Finnhub, 100 QPM Reddit). Prevents source bans and ensures reliable data in production.
📊
Deterministic Metrics
All 14 portfolio metrics are computed via pandas/numpy — no LLM inference in the calculation path. LLM is used only for narration and synthesis, never for arithmetic.
🧠
Checkpointed State
MemorySaver persists graph state at every node boundary. Enables resume after HITL pause, retry after failure, and full conversation history for the research chat.
